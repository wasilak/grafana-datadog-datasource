# Package Manager

## Use Yarn for Package Management

This project uses **Yarn** as the package manager, not npm.

### Commands

- ✅ `yarn install` - Install dependencies
- ✅ `yarn build` - Build the project
- ✅ `yarn test` - Run tests
- ✅ `yarn dev` - Start development mode
- ✅ `yarn lint` - Run linting

- ❌ `npm install` - Don't use npm
- ❌ `npm run build` - Don't use npm
- ❌ `npm test` - Don't use npm

### Why Yarn

- Project has `yarn.lock` file
- Consistent with existing project setup
- Faster and more reliable dependency resolution

### When to Use

- Always use `yarn` commands for package management
- When suggesting commands to users, use `yarn` syntax
- When writing documentation, reference `yarn` commands
- When creating build scripts, use `yarn` commands
